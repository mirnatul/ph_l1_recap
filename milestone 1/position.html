CSS position

  1. static: default
  2. relative: not left its own place, move relates to its current position
  3. absolute: left its position, will move based on the root, but if it's parent have a position mentioned it will move to its perspective
  4. fixed: fixed to viewport, not have it's own position
  5. sticky: If the parent is not scrollable, the sticky behavior will apply relative to the viewport, but if it has scrolling behavior, it will stick relative to the parent's boundaries.
